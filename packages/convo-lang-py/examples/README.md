# Convo-Lang Examples

This directory contains example projects demonstrating different ways to use
Convo-Lang from Python.

The examples are grouped by purpose and complexity.

## Structure

### 01_basics
Simple, focused examples demonstrating individual Convo-Lang features.
Planned but not yet added.

### 02_patterns
Reusable patterns and agent pipelines built with Convo-Lang.
These examples show how multiple `.convo` agents can be orchestrated from Python.

### 03_apps
End-to-end application examples.
Planned for future releases.
