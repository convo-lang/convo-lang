# Convo-Lang CLI - Linux Arm64 (non musl)
This is the native binary for the Convo-Lang CLI for Linux Arm64 (non musl) for distros such as Ubuntu
and Arch that use glibc. Light weight distros such as Alpine should use the musl binary.

see https://github.com/convo-lang/convo-lang/blob/main/packages/cli/README.md for more information.

This binary was built using `bun build --compile`
