# Convo-Lang CLI - Linux x64
This is the native binary for the Convo-Lang CLI for Linux x64 for light weight distros such as
Alpine that use libc. Distros such as Ubuntu and Arch that use glibc should use the non musl binary.

see https://github.com/convo-lang/convo-lang/blob/main/packages/cli/README.md for more information.

This binary was built using `bun build --compile`
