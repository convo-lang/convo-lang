# GPT-5 Reasoning Effort and Verbosity
The `.convo` files in this directory demonstrate using different combinations of GPT-5 reasoning
effort and response verbosity.

The [Convo-Lang VSCode Extension](https://marketplace.visualstudio.com/items?itemName=IYIO.convo-lang-tools) to configure and run the prompts.

Search "Convo-Lang" in the extensions panel to install

![reasoning-effort](https://raw.githubusercontent.com/convo-lang/convo-lang/refs/heads/main/examples/convo/reasoning-effort/reasoning-effort.png)
